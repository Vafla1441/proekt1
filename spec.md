# Пояснительная записка
## Проект "Задачник на Python и Qt"

**Автор:** Никифоров Федор 9 класс

**Цель:** Создать программу для сохранения и отслеживания своих задач.

**Технологии:** 
- Python - основной язык
- Qt (PyQt) - для интерфейса
- SQLite - для хранения задач

**Функции:**
- Просмотр задач по категория
- Редактор категорий
- Сортировка по категориям
- Хранение задач

**Результат:** Рабочее приложение для отслеживания личных задач.
