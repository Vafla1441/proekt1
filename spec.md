# Пояснительная записка
## Проект "Задачник на Python и Qt"

**Автор:** ученик 9 класса

**Цель:** Создать программу для изучения программирования с задачами и проверкой решений.

**Технологии:** 
- Python - основной язык
- Qt (PyQt/PySide) - для интерфейса
- SQLite - для хранения задач

**Функции:**
- Просмотр задач по темам
- Редактор кода с проверкой
- Система тестирования решений
- Хранение прогресса

**Результат:** Рабочее приложение для помощи в изучении Python.
